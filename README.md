# Meu Primeiro Projeto

Estou aprendendo Java e meu primeiro projeto será um sistema para gerenciar gastos com cartão de crédito chamado GestaoCartao.

GestaoCartao 💳
O GestaoCartao é uma aplicação robusta desenvolvida para facilitar o controle e a organização de gastos em cartões de crédito. O sistema permite o monitoramento detalhado de transações, integrando cadastros de clientes, estabelecimentos e movimentações financeiras para oferecer uma visão clara da saúde financeira do usuário.

🚀 Funcionalidades Principais
O sistema é composto por diversos módulos de cadastro e gerenciamento:

Gestão de Clientes: Cadastro completo dos usuários do sistema.

Controle de Lojas: Registro dos estabelecimentos onde as compras são realizadas.

Módulo Financeiro: Lançamento de faturas, parcelamentos e acompanhamento de limites.

Catálogo de Produtos: Registro de itens adquiridos para maior detalhamento do consumo.

Cadastros Auxiliares: Tabelas de apoio para otimizar o armazenamento e a integridade dos dados.

🛠️ Tecnologias Utilizadas
O projeto utiliza uma stack moderna e amplamente adotada no mercado:

Backend: Java (Linguagem principal para lógica de negócios e API).

Frontend: HTML5, CSS3 e JavaScript (Interface do usuário responsiva e dinâmica).

Banco de Dados: PostgreSQL (Banco relacional para armazenamento seguro e escalável).

🏗️ Arquitetura do Sistema
O sistema foi planejado para garantir a persistência correta das informações:

Camada de Visão: Interfaces limpas para entrada de dados.

Camada de Serviço: Regras de negócio implementadas em Java.

Camada de Dados: Consultas otimizadas no PostgreSQL para relatórios financeiros.

🔧 Como Executar o Projeto
Pré-requisitos
JDK 17 ou superior.
PostgreSQL instalado e configurado.
IDE de sua preferência (IntelliJ, Eclipse ou VS Code).

Passos para Instalação
Clone o repositório:

Bash
git clone https://github.com/seu-usuario/GestaoCartao.git
Configure o Banco de Dados:

Crie um banco de dados chamado gestao_cartao.
Atualize as credenciais no arquivo de configuração do projeto (ex: application.properties).

Execute a Aplicação:
Compile e rode o projeto através da sua IDE ou via terminal.
